# coren-starter-kit

[![Greenkeeper badge](https://badges.greenkeeper.io/Canner/coren-starter-kit.svg)](https://greenkeeper.io/)

## Usage

```bash
npm install
```

### development

```bash
node app.js
```
after file is built
```bash
npm run coren-dev
```

### production

```bash
npm run coren-production
```
